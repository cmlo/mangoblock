## Using

```bash
# Clone project
git clone --recurse-submodules https://github.com/mangoBlock/mangoBlock-desktop.git

# Install electron-rebuild
npm install --save-dev electron-rebuild

# Install the dependencies 
npm install

# Run program
npm start
```
